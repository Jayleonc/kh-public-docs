Knowledge Hub 是一个面向企业知识场景的智能平台。它把文档、权限、检索和 AI 对话放在同一套系统里，让外部用户可以通过一个简单的 Web Chat 入口了解系统能力，也可以在被授权的范围内查询文档内容。

这个站点本身由 Knowledge Hub 托管。你现在看到的内容来自 Git 仓库同步，系统会把 Markdown 文档同步到知识空间，再通过检索和 Agent 问答提供可引用的回答。

## 你可以怎么使用

可以直接阅读左侧文档，也可以点击聊天入口提问。适合提问的问题包括：

- Knowledge Hub 解决什么问题？
- 它和普通知识库或 Chatbot 有什么区别？
- RAG 和 Agent 是怎么配合的？
- Git Sync 为什么适合维护客户文档？
- 多租户和权限隔离是怎么设计的？

## 回答边界

对外 Demo 的回答应该只基于这个公开文档集。没有写进公开文档的内部实现细节、客户信息、部署凭据和私有配置，都不应该出现在回答里。
